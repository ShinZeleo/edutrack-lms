# Screenshots Directory

Tempatkan screenshot aplikasi di folder ini.

## Screenshot yang Diperlukan

1. **homepage.png** - Screenshot halaman utama/homepage
2. **admin-dashboard.png** - Screenshot dashboard admin
3. **course-catalog.png** - Screenshot katalog kursus
4. **lesson-view.png** - Screenshot tampilan lesson
5. **user-management.png** - Screenshot manajemen user

## Format

- Format: PNG atau JPG
- Resolusi: Minimal 1920x1080 (Full HD)
- Ukuran file: Maksimal 2MB per gambar
- Nama file: Gunakan kebab-case (contoh: `admin-dashboard.png`)

## Cara Menambahkan Screenshot

1. Ambil screenshot dari aplikasi yang sedang berjalan
2. Simpan dengan nama sesuai daftar di atas
3. Place di folder `docs/screenshots/`
4. Commit dan push ke repository
